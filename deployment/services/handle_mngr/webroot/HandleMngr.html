NAME

Bio::KBase::HandleMngr::Client

DESCRIPTION

The HandleMngr module provides an interface for the workspace service to make handles sharable. When the owner shares a workspace object that contains Handles, the underlying shock object is made readable to the person that the workspace object is being shared with.

is_readable

  $return = $obj->is_readable($token, $nodeurl)
Parameter and return types
$token is a string
$nodeurl is a string
$return is an int
Description

The is_readable function will return true if the underlying shock object is readable by the owner of the token. The token is passed by the client.

add_read_acl

  $obj->add_read_acl($hids, $username)
Parameter and return types
$hids is a reference to a list where each element is a HandleId
$username is a string
HandleId is a string
Description

TYPES

HandleId

Description

The add_read_acl functions will update the acl of the shock node that the handle references. The function is only accessible to a specific list of users specified at startup time. The underlying shock node will be made readable to the user requested.

Definition
a string
