Description

given a list of sample ids and feature ids and the string of what type of numerical interpretation it returns a LabelDataMapping {sampleID}->{featureId => value}}. If sample id list is an empty array [], all samples with that feature measurment values will be returned. If feature list is an empty array [], all features with measurment values will be returned. Both sample id list and feature list can not be empty, one of them must have a value. Numerical_interpretation options : 'FPKM', 'Log2 level intensities', 'Log2 level ratios' or 'Log2 level ratios genomic DNA control'
